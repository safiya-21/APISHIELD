# API Shield – Secure API System

## Overview
A backend system built in Core Java to secure APIs using authentication and rate limiting.

## Features
- API Key Authentication
- Rate Limiting (prevents abuse)
- Request Logging

## How it works
1. Client sends request with API key
2. System validates key
3. Rate limiter checks request frequency
4. Response is processed or blocked

## Tech Stack
- Java
- OOP principles
